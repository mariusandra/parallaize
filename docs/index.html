1. Install the package

Download the .deb that matches your host architecture, then install it.

sudo apt install ./parallaize_0.1.0-1_amd64.deb
# or
sudo apt install ./parallaize_0.1.0-1_arm64.deb

2. Review config and boot services

The package ships the env file and systemd units. Review the env file first.

sudoedit /etc/parallaize/parallaize.env
sudo systemctl start parallaize.service
sudo systemctl start parallaize-caddy.service
